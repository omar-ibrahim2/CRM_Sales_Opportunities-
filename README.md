# Data Analysis Project (SQL)

## Project Overview
This project focuses on analyzing business data using SQL to answer key business
questions and extract meaningful insights that support decision-making.
The analysis helps stakeholders understand sales performance, trends, and key revenue drivers.

## Business Questions
- How is each sales team performing compared to the rest?
- Are there any quarter-over-quarter sales trends?
- Do any products have better win rates?
- What is the average sales value per sector?
- What is the relationship between company size and revenue?
- How do regional offices compare in terms of sales and number of employees?

## Tools & Technologies
- SQL
- Advanced SQL (Joins, Aggregations)
- DataCamp DataLab

## Key Steps
- Data exploration using SQL queries
- Data cleaning and transformation
- Exploratory Data Analysis (EDA)
- Insight generation and interpretation

## Key Insights
- Identified top contributors to total revenue
- Detected trends and seasonal patterns
- Highlighted performance gaps across categories
